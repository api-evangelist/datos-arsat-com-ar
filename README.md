# Datos Abiertos ARSAT (datos-arsat-com-ar)

Datos Abiertos ARSAT is a government agency open-data portal for Argentina running **CKAN**, exposing a standardized catalog API over ~13 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/datos-arsat-com-ar/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=datos-arsat-com-ar-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.7.6

## API
- **Datos Abiertos ARSAT CKAN Action API** — base `https://datos.arsat.com.ar/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://datos.arsat.com.ar)
- Live endpoint verified: `https://datos.arsat.com.ar/api/3/action/status_show` · ~13 datasets
